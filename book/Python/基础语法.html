基础语法

标识符

Python中标识符的第一个字符必须是字母或者下划线_，其余部分可由字母、下划线或数字组成。标识符是大小写敏感的。在 Python 3中，非ASCII标识符也是允许的了。

保留字

即关键字，不能用作任何标识符名称，因为会发生歧义。Python标准库提供了一个keyword模块，输出当前版本所有的关键字。

多行语句

通常一条语句一行写完，但如果语句过长，可以使用反斜杠来书写多行语句。

另外，[] {}或()中的多行语句，不需要使用反斜杠。

Python可以在同一行输入多个语句，语句间使用分号间隔。

数据类型

Python中数字有四种类型：整数、长整数、浮点数和复数。

  • int(整数)，100
  • long(长整数)，比较大的整数
  • float(浮点数)，3.1415
  • complex(复数)，如1+2j

字符串

Python使用单引号或双引号表示字符串，两者完全相同。使用三引号可指定一个多行字符串。字符串是不可变的。

使用转义符\表示特殊字符，比如\n换行。通过在字符串前加r或R，可阻止转义，比如r”a fox jump over there \n”会显示\n，而不是换行。

python允许处理unicode字符串，加前缀u或U， 如 u”this is an unicode string”。

空行

函数之间或类的方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分隔，以突出函数入口的开始。空行与代码缩进不同，空行并不是Python语法的一部分。书写时不插入空行，Python解释器运行也不会出错。但是空行的作用在于分隔两段不同功能或含义的代码，便于日后代码的维护或重构。

代码组

缩进相同的一组语句构成一个代码块，称为代码组。比如if、while、def和class这样的复合语句，首行以关键字开始，以冒号( : ) 结束，该行之后的一行或多行代码构成代码组。

将首行及后面的代码组称为一个子句(clause)。

导入模块

使用import或者from...import来导入相应的模块或者函数。

  • import somemodule 导入整个模块
  • from somemodule import somefunction 导入某个模块中的某个函数
  • from somemodule import firstfunc, secondfunc, thirdfunc 导入某个模块中的多个函数
  • from somemodule import * 导入某个模块中的全部函数
